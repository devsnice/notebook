Notebook
=============
The small responsive notebook for writting new notes

Installing
----------

**git shell**
```shell
git clone https://github.com/devsnice/notebook/ notebook
```

**nodejs shell**
```shell
cd notebook
gulp
```

Browsers
----------

    Firefox 3.5+
    Chrome 8+
    IE 9+
    Opera 11.5+
    Safari 5.0+
    
    
Writing with
----------
*gulp, require.js, sass*
